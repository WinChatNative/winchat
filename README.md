# WinChat
WinChat은 MQTT 프로토콜 기반 서버와 클라이언트로 구성됩니다. 클라이언트는 WPF 기반 채팅 프로그램으로 서버는 미정입니다.

오픈카카오톡 채널의 `Windows Native 개발자 모임` 에서 파생된 토이프로젝트입니다.
